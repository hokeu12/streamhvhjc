• Directly paste any Cloudinary video URL (e.g., https://res.cloudinary.com/your-cloud/video/upload/v123456/sample.mp4) and click Load video.
• You can also pass the video URL via query parameter: ?video=YOUR_CLOUDINARY_URL
• Supports MP4, WebM, HLS (.m3u8) if the browser supports it. For best results use an mp4 delivery URL from Cloudinary.
